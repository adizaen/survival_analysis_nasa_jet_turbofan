PREDICTIVE MAINTENANCE OF JET ENGINE

About

  • Predictive Maintenance of Jet Engine serves as an early detection tool for engine damage.
  • This application uses a pre-trained Random Forest model.
  • The model used has a high accuracy up to 96%.
  • Fill in the form below to enter sensor data and use the TAB TAB key to move the field.
  • Results will show the condition of the engine whether NORMAL or FAILURE based on the model that has been trained.
